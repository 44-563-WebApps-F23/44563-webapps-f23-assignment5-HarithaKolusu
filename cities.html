Cities names breaks first

As you decrease the screen size, you'll observe that the city names will wrap to the next line before the letters. This is due to Bootstrap's responsive design, which adapts column sizes based on the screen's width. When the browser window becomes smaller, the columns with the "col-md" class which is designed for medium-sized screens will stack vertically while the "col-sm-2" columns will retain their horizontal layout for a more extended period.
